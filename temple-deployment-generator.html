📚 詳細部署步驟

步驟 1：建立GitHub倉庫

1. 前往 GitHub.com 並登入

2. 點擊右上角「+」→「New repository」

3. Repository name: 填寫宮廟資訊後顯示

4. 設為 Public，點擊「Create repository」

步驟 2：上傳檔案

1. 下載本頁面產生的所有檔案

2. 在GitHub倉庫頁面點擊「uploading an existing file」

3. 將所有檔案拖拽上傳

4. 填寫 Commit message：「初始化宮廟記帳軟體」

5. 點擊「Commit changes」

步驟 3：啟用GitHub Pages

1. 在倉庫頁面點擊「Settings」

2. 左側選單找到「Pages」

3. Source 選擇「Deploy from a branch」

4. Branch 選擇「main」

5. 點擊「Save」

步驟 4：完成！

🎉 恭喜！您的宮廟軟體已上線
