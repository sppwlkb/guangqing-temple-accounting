# 🏛️ 宮廟快速記帳軟體 - 完整使用指南

## 📋 軟體簡介

**宮廟快速記帳軟體**是專為台灣宮廟設計的通用記帳管理系統，具備以下特色：

- ✅ **多宮廟支援** - 每個宮廟擁有獨立的資料空間
- 🔐 **個資保護** - 完全符合台灣個人資料保護法
- 💾 **彈性儲存** - 支援本地儲存或自建雲端資料庫
- 🎨 **客製化** - 可自訂宮廟資訊、主題和設定
- 📱 **跨平台** - 支援電腦、平板、手機使用

## 🚀 快速開始

### 第一次使用

1. **開啟軟體**
   - 訪問：https://sppwlkb.github.io/guangqing-temple-accounting/
   - 系統會自動顯示「宮廟記帳軟體初始設定」精靈

2. **完成基本設定**
   ```
   📝 必填項目：
   • 宮廟名稱
   • 個資保護負責人姓名
   
   🔧 選填項目：
   • 宮廟地址
   • 聯絡電話
   • 主祀神明
   • 負責人聯絡電話
   ```

3. **選擇資料儲存方式**
   - **本地儲存（推薦）**：免費，資料存在瀏覽器中
   - **Firebase雲端**：需自行申請Firebase帳號

4. **完成設定**
   - 點擊「完成設定」
   - 系統會自動重新載入並應用您的設定

## 💾 資料儲存方案比較

| 方案 | 優點 | 缺點 | 適用對象 |
|------|------|------|----------|
| **本地儲存** | • 完全免費<br>• 無需額外設定<br>• 資料完全在自己掌控 | • 僅限單一裝置<br>• 需定期備份 | 小型宮廟<br>單一使用者 |
| **Firebase雲端** | • 多裝置同步<br>• 自動備份<br>• 高可用性 | • 需自行申請帳號<br>• 可能產生費用 | 中大型宮廟<br>多人使用 |

## 🔐 個資保護功能

### 自動保護機制

1. **資料加密**：所有敏感個人資料自動加密存儲
2. **存取控制**：可設定不同使用者的權限等級
3. **審計日誌**：完整記錄所有資料存取行為
4. **同意書管理**：符合個資法的同意與撤回機制
5. **自動清理**：到期資料自動清理（預設5年）

### 個資保護功能入口

- **個資保護中心**：`privacy-quick-access.html`
- **隱私政策**：`privacy-policy.html`
- **管理員儀表板**：`privacy-dashboard.html`

## 👥 多宮廟使用方式

### 資料隔離機制

每個宮廟的資料完全獨立：

```
宮廟A的資料 → temple-ABC123-records
宮廟B的資料 → temple-XYZ789-records
宮廟C的資料 → temple-DEF456-records
```

### 新宮廟加入步驟

1. **清除瀏覽器資料**（如果之前使用過）
   - Chrome：設定 → 隱私權和安全性 → 清除瀏覽資料
   - 選擇「Cookie和其他網站資料」和「已快取的圖片和檔案」

2. **重新訪問軟體**
   - 系統會自動顯示設定精靈
   - 輸入新宮廟的資訊

3. **完成設定**
   - 新宮廟會獲得獨立的資料空間

## 🛠️ 進階設定

### Firebase 雲端設定

如果選擇使用Firebase雲端儲存：

1. **申請Firebase帳號**
   - 訪問：https://console.firebase.google.com/
   - 使用Google帳號登入

2. **建立新專案**
   - 點擊「建立專案」
   - 輸入專案名稱（建議：您的宮廟名稱）
   - 完成建立流程

3. **取得配置資訊**
   - 在專案總覽中點擊「網頁」圖示
   - 複製配置資訊中的 `apiKey` 和 `projectId`

4. **在軟體中設定**
   - 選擇「Firebase雲端儲存」
   - 填入 API Key 和 Project ID

### 主題自訂

支援三種內建主題：
- **temple-gold**：金色主題（預設）
- **temple-red**：紅色主題
- **temple-blue**：藍色主題

### 員工權限設定

可在配置中自訂員工角色：
```javascript
staff: [
    {
        id: 'ADMIN001',
        name: '系統管理員',
        department: '管理部',
        role: 'admin',        // admin, data_manager, believer_manager
        level: 'high'         // high, medium, low, none
    }
]
```

## 🔧 常見問題

### Q1: 如何備份資料？
**A:** 
- 本地儲存：使用瀏覽器匯出功能或定期截圖
- Firebase：資料自動備份到雲端
- 建議定期匯出重要報表

### Q2: 多人使用時如何管理權限？
**A:** 
- 在宮廟配置中設定員工資訊
- 使用個資保護儀表板管理存取權限
- 每個使用者都有完整的操作記錄

### Q3: 如何確保個資安全？
**A:** 
- 系統自動加密所有敏感資料
- 定期執行合規檢查
- 遵循個資保護法的所有要求
- 提供完整的審計追蹤

### Q4: Firebase 會產生費用嗎？
**A:** 
- Firebase 提供免費額度
- 小型宮廟通常不會超過免費額度
- 詳細費用請參考 Firebase 官方說明

### Q5: 如何更換宮廟資訊？
**A:** 
- 進入系統後，在設定中尋找「重置配置」
- 或清除瀏覽器資料後重新設定
